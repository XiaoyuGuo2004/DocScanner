DocScanner是一个基于 Python + OpenCV 的桌面端文档扫描工具。它能够自动检测纸张边缘，进行透视矫正，并应用图像增强算法，将手机拍摄的文档照片转换为类似扫描仪效果的清晰图片。

✨ 功能特性

•   自动边缘检测：自动识别图像中的文档轮廓。

•   透视矫正：将歪斜拍摄的文档拉直为正视图。

•   图像增强：

    ◦   CLAHE（限制对比度自适应直方图均衡化）提升对比度。

    ◦   锐化滤镜增强文字清晰度。

    ◦   非局部均值降噪去除噪点。

•   多格式输出：一次性生成原图矫正版、灰度增强版、二值化版。

•   图形化操作：无需命令行，通过系统文件选择框即可操作。

📦 环境依赖

•   Python 3.7+

•   OpenCV-Python

•   NumPy

•   Tkinter (通常随 Python 自带)

🚀 安装步骤

1.  克隆或下载代码到本地
    
2.  安装依赖库
    建议使用虚拟环境。
    pip install opencv-python numpy
    
    > 注意：如果使用的是 Linux 系统，可能需要安装 Tkinter 依赖：
    > sudo apt-get install python3-tk

💡 使用方法

直接运行 scan.py 脚本：
python scan.py


运行后：
1.  会自动弹出文件选择窗口。
2.  选择一张包含文档的照片（支持 JPG, PNG, BMP），本项目中提供了一个可用来测试的jpg文件，该文件来自网络，侵权必删。
3.  程序会自动处理并在当前目录下生成 scans 文件夹。
4.  处理完成后会弹出提示框，告知输出路径。

📂 输出说明

程序会在项目根目录下创建 scans/ 文件夹，并根据时间戳生成子文件夹。

例如：scans/your_image_20260121_143022/

该文件夹内包含以下文件：

文件名 描述

01_original_warped.jpg 经过透视矫正后的原始彩色图

02_enhanced_grayscale.jpg 经过增强处理的灰度图

03_binary.jpg OTSU 自适应阈值二值化图（黑白）

RECOMMENDED_enhanced.jpg 👈 推荐使用的增强版图片 (最清晰)

⚠️ 注意事项

1.  拍摄建议：为了获得最佳效果，请确保拍摄时文档边缘完整出现在画面中，光线尽量均匀。
2.  检测失败：如果文档边缘不明显（如与背景色差小），程序会跳过矫正步骤，直接对原图进行增强处理。
3.  中文路径：虽然 Tkinter 和 OpenCV 通常支持中文路径，但在某些旧版本环境下可能存在编码问题，建议使用英文路径。

🔧 参数调整

如果你熟悉代码，可以在 enhance_image 函数中调整以下参数以适应不同的文档类型：

•   clipLimit=3.0：控制对比度增强的强度（越大对比越强烈）。

•   tileGridSize=(8,8)：控制局部对比度的网格大小。

•   kernel：锐化卷积核，数值越大锐化效果越强（注意不要过曝）。
