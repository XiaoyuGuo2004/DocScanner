# DocScanner
A simple docscanner.
文档扫描工具 使用说明书

1. 简介
文档扫描工具是一款本地运行的文档扫描增强工具，能将手机或相机拍摄的倾斜、歪斜文档照片自动转换为高质量扫描效果。支持自动边缘检测、透视校正（斜放自动扶正）、对比度增强、锐化等功能。

3. 运行环境
操作系统：Windows 10 / Windows 11
无依赖要求、无硬件要求，适用于大多数现代计算机。

4. 安装与启动
无需安装任何依赖，下载后直接解压缩到任意目录。
双击文件夹中的文档扫描工具.exe 即可启动。

5. 使用方法
启动软件后，会弹出终端，无需理会，稍等几秒后会自动弹出文件选择对话框。
选择需要处理的文档照片（支持 .jpg、.jpeg、.png、.bmp 等格式）。
注：文件夹中提供了test.jpg 供测试使用，您也可以使用自己的照片进行测试。
点击“打开”，软件开始自动处理。
处理完成后弹出“成功”提示框，并自动在程序所在目录下生成 scans 文件夹。

6. 输出文件说明
处理完成后，在 scans\图片名_时间戳\ 文件夹中生成以下文件：
    01_original_warped.jpg —— 透视校正后的原始色彩图
    02_enhanced_grayscale.jpg —— 增强后的灰度图
    03_binary.jpg —— 二值化黑白图
    RECOMMENDED_enhanced.jpg —— 推荐使用的最佳增强效果图

7. 注意事项
拍照时尽量让文档占据画面主要区域，背景简洁。
若文档边缘不明显或背景复杂，可能无法完美自动检测，此时会直接对原图进行增强。
输出文件夹会按时间自动命名，不会覆盖之前文件。

8. 常见问题
弹出“无法读取图片”：确认图片路径正确且未被其他程序占用。
处理速度较慢：属于正常情况（OpenCV 计算量较大），几秒内即可完成。

9. 后备方案
如果.exe程序最终因各种意外原因无法运行，您可以使用python运行scanner.py，前提是您已安装Python环境和OpenCV库。
